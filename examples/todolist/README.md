# ToDoList Example App

This is a minimal ToDoList example for the JMiniApp framework.

## How to run locally
1. Open `examples/todolist/index.html` in a browser to see a minimal demo.
2. Business logic is in `examples/todolist/main.js`.

## Files
- `main.js` - example logic (TodoList and TodoItem).
- `index.html` - minimal UI for testing.
