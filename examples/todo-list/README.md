# Todo List Example

Ejemplo completo de lista de tareas con JMiniApp, usando adaptador JSON personalizado para persistencia.

## Características
- Añadir tareas
- Listar tareas
- Eliminar tareas (marcar como completadas)
- Persistencia en `todo-state.json`

## Cómo ejecutar
1. `cd examples/todo-list`
2. `mvn clean package`
3. `mvn exec:java`
